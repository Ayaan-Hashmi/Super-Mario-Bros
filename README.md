## Running

* $ pip install -r requirements.txt
* $ python main.py

## Standalone windows build

* $ pip install py2exe
* $ python compile.py py2exe

## Controls

* Left: Move left  // Left Arrow
* Right: Move right  //Right Arror
* Space: Jump       //Up Arror
* Shift: Boost   
* Left/Right Mouseclick: Extra enemies and coins
